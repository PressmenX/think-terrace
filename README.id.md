# ThinkTerrace

[![Repo](https://img.shields.io/badge/vercel-live_demo-darkgreen?logo=vercel)](https://github.com/PressmenX/think-terrace) [![Repo](https://img.shields.io/badge/github-repo-blue?logo=github)](https://github.com/PressmenX/think-terrace) </br>
ThinkTerrace adalah aplikasi e-commerce sederhana berbasis web yang dikhususkan untuk lini produk laptop legendaris, Lenovo ThinkPad. Proyek ini dibangun menggunakan Vanilla JavaScript dengan pendekatan arsitektur modern untuk mendemonstrasikan manajemen state, filter data dinamis, dan sistem keranjang belanja.

## 🚀 Main Feature
- Dynamic Prodcut Catalog : Menampilkan daftar laptop ThinkPad yang diambil dari data JSON.
- Filtering Feature : Pengguna dapat memfilter produk berdasarkan Seri (X, T, L, E) dan rentang harga secara bersamaan.
- Sorting Feature : Mendukung pengurutan berdasarkan harga (termurah/termahal) dan rating tertinggi.
- Cart Drawer: Menambah atau menghapus produk dari keranjang, Menyesuaikan kuantitas produk secara langsung di dalam keranjang, Penghitungan otomatis untuk subtotal per item dan total keseluruhan.
- Data Persistence : Menggunakan localStorage agar data keranjang belanja tetap tersimpan meskipun halaman dimuat ulang.
- Responsive Design : Antarmuka yang optimal untuk perangkat mobile hingga desktop menggunakan CSS Grid dan Flexbox.

## ⚡Installation & How to Run

1. Clone repositori ini
    ```
   git clone https://github.com/username/think-terrace.git
    ```
2. Masuk ke direktori proyek
    ```
    cd think-terrace
    ```
3. Instal dependensi
   ```
   npm install
   ```
4. Jalankan server pengembangan
   ```
   npm run dev --open
   ```
