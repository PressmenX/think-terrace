# ThinkTerrace

[![Repo](https://img.shields.io/badge/vercel-live_demo-darkgreen?logo=vercel)](https://github.com/PressmenX/think-terrace) [![Repo](https://img.shields.io/badge/github-repo-blue?logo=github)](https://github.com/PressmenX/think-terrace) </br>
ThinkTerrace is a simple web-based e-commerce application dedicated to the legendary Lenovo ThinkPad laptop product line. This project is built using Vanilla JavaScript with a modern architectural approach to demonstrate state management, dynamic data filtering, and a shopping cart system.

## 🚀 Main Features
- Dynamic Product Catalog: Displays a list of ThinkPad laptops retrieved from JSON data.
- Filtering Feature: Users can filter products by Series (X, T, L, E) and price range simultaneously.

- Sorting Feature: Supports sorting by price (cheapest/most expensive) and highest rating.
- Cart Drawer: Add or remove products from the cart, adjust product quantities directly in the cart, and automatically calculate subtotals per item and the grand total.
- Data Persistence: Uses localStorage to retain shopping cart data even when the page is reloaded.
- Responsive Design: An optimal interface for both mobile and desktop devices using CSS Grid and Flexbox.

## ⚡Installation & How to Run

1. Clone this repository 
``` 
git clone https://github.com/username/think-terrace.git 
```
2. Go to the project directory 
``` 
cd think-terrace 
```
3. Install dependencies 
``` 
npm install 
```
4. Run the development server 
``` 
npm run dev --open 
```
